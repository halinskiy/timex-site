# Timex — website

Marketing site for [Timex](https://github.com/halinskiy/timex), a minimal, keyboard-driven time tracker for macOS.

Built with Vite + React + TypeScript + Tailwind, using Relume components.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build      # → dist/
npm run preview
```

## Deploy

Published to GitHub Pages from the `gh-pages` branch at
`https://halinskiy.github.io/timex-site/`. Rebuild `dist/` and push its
contents to `gh-pages`.
